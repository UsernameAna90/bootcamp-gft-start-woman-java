# Comandos básicos Git:

- git clone < link >: clona um repositório do git hub por meio de sua respectiva url.

- git init: inicializa o git em um repositório.

- git status: verifica o atual status do do diretório de trabalho e da área de staging.

- git add < * >: inclui atualizacoes realizadas em um arquivo.

- git commit -m " < mensagem > ": salva atualizacoes realizadas em um arquivo, junto a uma mensagem relatando as alteracoes feitas.

- git push origin main: empurra as atualizacoes realizadas em um arquivo para o repositorio no GitHub.
